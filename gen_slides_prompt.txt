    * **A frequent LLM error is confusing mathematical comparison operators (`>`, `<`) with curly braces (`}}`, `{{`). This error MUST NOT occur in the output.**
    * **Rule 1:** Mathematical 'greater than' MUST ALWAYS be written as `$ > $` within math mode. NEVER use `}}`.
        * **INCORRECT EXAMPLE TO AVOID:** `$r }} R$` or `$C_1 }} C_2$`
        * **CORRECT USAGE:** `$r > R$`, `$C_1 > C_2$`
    * **Rule 2:** Mathematical 'less than' MUST ALWAYS be written as `$ < $` within math mode. NEVER use `{{`.
        * **INCORRECT EXAMPLE TO AVOID:** `$C_1 {{ C_2$` or `$R {{ r$`

    * **Rule 3:** Standard curly braces `{{` and `}}` are ONLY for LaTeX grouping (e.g., `\\section{{Title}}`, `x^{{2}}`) or set notation (`\\{{ x | x > 0 \\}}`). They are NEVER substitutes for comparison operators.
